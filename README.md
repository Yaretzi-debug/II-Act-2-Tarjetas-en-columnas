Actúa como un desarrollador experto en Flutter. Necesito crear la interfaz de una aplicación móvil de repostería llamada 'Fresh Tarts' con las siguientes especificaciones:

1. Estructura de la UI (Basado en imagen de referencia):
Diseña una pantalla con una columna vertical que contenga exactamente 5 tarjetas. Cada tarjeta debe replicar el diseño de 'tarjetas en filas' de la imagen de referencia (estilo Microsoft/Edu4rdSHL):

Contenedor: Fondo blanco, bordes redondeados (Card) y una sombra sutil.

Distribución: A la izquierda la imagen del postre (cuadrada o circular) y a la derecha el bloque de texto.

Tipografía de las tarjetas: El nombre del producto arriba y un subtítulo en negritas debajo (siguiendo el estilo visual exacto de la imagen).

2. Concepto Visual y Colores:

Temática: Repostería de conejos.

Paleta de colores: Fondo de la app en gris pastel y detalles (botones o AppBar) en rosado pastel.

Fuentes: El título de la app 'Fresh Tarts' debe ir en letras cursivas (Script) en la parte superior.

3. Datos de las 5 Tarjetas (Usar estas URLs de imagen):

Macaron Bunny: https://raw.githubusercontent.com/Yaretzi-debug/imagenes-para-flutter-6to-I-fecha-11-feb-2026/refs/heads/main/macaron.jfif

Rabbit Cake: https://raw.githubusercontent.com/Yaretzi-debug/imagenes-para-flutter-6to-I-fecha-11-feb-2026/refs/heads/main/pastel.webp

Paw Cookie: https://raw.githubusercontent.com/Yaretzi-debug/imagenes-para-flutter-6to-I-fecha-11-feb-2026/refs/heads/main/galleta.jpg

Bunny Party Box: https://raw.githubusercontent.com/Yaretzi-debug/imagenes-para-flutter-6to-I-fecha-11-feb-2026/refs/heads/main/galletas.jfif

Fluffy Cupcake: https://raw.githubusercontent.com/Yaretzi-debug/imagenes-para-flutter-6to-I-fecha-11-feb-2026/refs/heads/main/cupcake.jpg

4. Requerimiento Técnico:
Entrégame el código de Flutter listo para copiar en main.dart, usando widgets como ListView o Column con SingleChildScrollView para mostrar las 5 tarjetas.
